## Methods

* Developed a scalable FastAPI backend integrating Playwright and BeautifulSoup for dynamic web data extraction
* Implemented concurrent scraping and adaptive parsing to handle JavaScript-rendered, anti-bot-protected e-commerce sites
* Modeled the system as an extensible data ingestion API for structured, machine-learning-ready product datasets
